python基础整理

python 基础语法

print 函数

print可以输出变量，数值，字符串

运用 chr 可以将数字在ASCII表中对应，从而输出 字符

也可以向文件中写入数据

连接输出

中间写一个加号可以连接两个字符串，字符串不可以和整数连接

Input函数(数据读入)

注意 input 里面要写提示词，最好是写上

input 函数默认读入的都是字符串类型

字符串类型切片

[l,r] 从 l 开始切到 r ，不包含 r

[l,r] l 表示开始位置，如果没有 l，则代表从字符串开头开始切片

r 表示结束位置且不包含，如果没有 r ，代表一直切到字符串结尾

字符串的三种操作

(1) + 用于两种字符串的拼接

(2) s1 * 2 表示将 s1 字符串重复了两次

(3) 判断当前子串是否在 s1 中出现过

布尔类型

总结：所有非 0 整数的布尔值都是 True，所有非空字符串的布尔值都是 True

eval 函数

用于去掉字符串最外侧的引号，并按照Python语句方式执行去掉引号后的字符串

eval 函数经常和 input 函数一起使用

序列和索引

序列是一个用于存储多个值的连续空间，每个值都对应一个整数的编号，称为索引

索引：(1) 正向递增索引 （2）反向递增索引

正向从0-n-1 反向从 -n - （-1）

切片操作的语法结构

序列[start: end :step]

start表示开始索引(包含)，end表示结束索引（不包含）， step表示步长，默认为 1

start或者end位置如果空的话，默认从开头开始或者到结尾结束

序列的相加相乘操作

列表(序列中的一种，序列的操作也可以在列表上使用)

是指一系列的按特定顺序排列的元素组成

是Python中内置的可变序列

在Python中使用 [] 定义列表，元素与元素之间使用英文的逗号分隔

列表中的元素可以是任意的数据类型

列表的创建方式：

(1) 使用 [] 直接创建列表

列表名 = [e1,e2,……….eN]

(2) 使用内置函数 list() 创建列表

列表名 = list(序列)

列表的删除：

del 列表名

列表的特有操作

列表的排序操作

(1) 列表对象的 sort 方法

lst.sort(key = None,reverse = False)

key表示排序的规则，reverse表示排序方式(默认升序)

(2) 内置函数sorted()

sorted(iterable,key = None,reverse = False)

iterable 表示的是排序的对象

使用sorted() 内置函数会产生一个新的列表，原列表没有改动

列表生成式

元组类型

元组是py中内置的不可变序列

在py中使用 () 定义元组，元素与元素之间使用英文的逗号分隔

元组中只有一个元素的时候，逗号也不能省略

元组的创建方式有两种

(1) 使用 () 直接创建元组

元组名 = (e1,e2,……eN)

(2) 使用内置函数 tuple() 创建元组

元组名 = tuple(序列)

删除元组: del 元组名

元组的访问和遍历

元组的生成式

字典类型（亦可使用序列的操作）

字典类型是根据一个信息查找另一个信息的方式构成了“键值对”，它表示索引用的键和对应的值构成的成对关系

可变数据类型不可作为字典中的键

字典的创建方式：

(1) d = {key1:value1,key2:value2……….}

(2) zip(lst1,lst2) lst1 , lst2 是两个列表，然后使用内置函数dict来将其转化为字典类型

字典元素的取值：

d[key] 或 d.get(key)

字典元素的遍历：

(1) 遍历出 key 与 value 的元组

(2) 分别遍历出 key 和 value

字典的相关操作方法

字典生成式：

d = {key:value for item in range}

d = {key:value for key,value in zip(lst1,lst2)}

集合类型

py 中的集合 与数学中的集合的概念一样

py 中的集合是一个无序的不重复元素序列

集合中只能存储不可变数据类型

py 中集合使用 {} 定义

与列表、字典一样，都是py中的可变数据类型

集合的创建方式

(1) 使用 {} 直接创建集合

s = {e1,e2,….eN}

(2) 使用内置函数 set() 创建集合

s = set(可迭代对象)

集合的删除：del 集合名

集合类型的操作符

交集，并集，差集，补集

集合的相关操作方法

s.add(x) ：如果 x 不在集合 s 中，则将 x 添加到集合 s

s.remove(x) ：如果 x 在集合中，将其删除，如果不在集合中，程序报错

s.clear()：清除集合中所有元素

字符串 和 正则表达式

字符串的编码和解码

bytes类型表示的是二进制数据

数据的验证

数据的验证是指程序对用户输入的数据进行合法性验证
