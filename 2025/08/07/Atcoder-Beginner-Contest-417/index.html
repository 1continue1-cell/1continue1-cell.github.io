A - A Substring

理解一下题意直接输出字符串即可

B - Search and Delete

这个题目开一个 map 记录一下即可，在 map 中的元素也是有序的

C - Distance Indicators

题目给定条件 j - i = a[i] + a[j]

那么可以将给定的条件进行移项，得到 j - a[j] = i + a[i] ,而又已经知道 j > i

则开两个哈希表，一边遍历数组一遍插入元素统计答案即可

D - Takahashi’s Expectation

本题是一道 dp ，只不过需要一些思考在里面

分析出状态 dp[i][j] 表示 从 i 开始往后都没有经历过，现在的心情值为 j ，最后的心情值

如果当前的心情 j <= pp[i].a(即当前礼物的价值) dp[i][j] = dp[i+1][j+pp[i].a]

否则的话 dp[i][j] = dp[i+1][max( 0, j - pp[i].b)]

由于 p a b 都在 500 范围内，所以 dp 数组的第二维最大是 1000

q 次查询，如果读入的数值 x 在 1000 以内 ans = dp[1][x]

否则 x 会一直递减到范围内，二分配合b元素的前缀和数组即可找到递减位置

E - A Path in A Dictionary

这个题目要寻找字典序最小的从 x 到 y 的联通路径，考虑使用并查集来解决

将一个点的所有出边从小到大排序后，从起点开始遍历节点所有出边，如果出边节点未被访问并且出边节点和 y 联通，那么则找到路径上的一个节点，按照这个思路一直处理下去，可以得到最终的答案路径

至于出边界点和 y 是否联通，则在每一次循环中将所有未被访问的节点对应的边连接，并查集判断即可
